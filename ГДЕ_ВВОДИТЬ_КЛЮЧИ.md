# 🔑 ГДЕ ВВОДИТЬ API КЛЮЧИ - Простая инструкция

## 🎯 На Render.com (пошагово):

### 1️⃣ Откройте Render.com
- Зайдите на https://render.com
- Войдите в аккаунт

### 2️⃣ Найдите ваш проект
- В списке проектов найдите `dns-automation` (или как вы назвали)
- **Нажмите на название проекта**

### 3️⃣ Откройте раздел Environment
- Слева в меню найдите **"Environment"** 
- Или сверху вкладка **"Environment"**
- **Нажмите на него**

### 4️⃣ Добавьте переменные
- Нажмите кнопку **"Add Environment Variable"** или **"Add Variable"**
- Появится форма с двумя полями: **Key** и **Value**

---

## 📝 Что вводить (скопируйте и вставьте):

### Переменная 1:
```
Key: CLOUDFLARE_EMAIL
Value: [ваш email от Cloudflare]
```

### Переменная 2:
```
Key: CLOUDFLARE_API_KEY
Value: [ваш Global API Key от Cloudflare]
```

### Переменная 3:
```
Key: REGISTRAR_API_URL
Value: https://api.ukraine.com.ua/v2
```

### Переменная 4:
```
Key: REGISTRAR_API_KEY
Value: [ваш API ключ от ukraine.com.ua]
```

---

## 🔍 Где взять ключи:

### Cloudflare Global API Key:
1. Зайдите: https://dash.cloudflare.com/profile/api-tokens
2. Прокрутите вниз до раздела **"API Keys"**
3. Нажмите **"View"** рядом с **"Global API Key"**
4. Введите пароль от Cloudflare
5. **Скопируйте ключ**

### Ukraine.com.ua API ключ:
1. Зайдите: https://www.ukraine.com.ua
2. Войдите в панель управления
3. Найдите раздел **"API"** или **"Настройки"** → **"API"**
4. Скопируйте **API ключ** (только ключ, без секрета)

---

## ✅ После добавления:

1. Нажмите **"Save Changes"** (или сохранится автоматически)
2. Render перезапустит приложение (2-3 минуты)
3. Откройте ваше приложение: `https://ваш-проект.onrender.com`

## ⚠️ Важно:

- **Key** - это название переменной (точно как написано выше)
- **Value** - это сам ключ/email (копируйте полностью, без пробелов)
- После добавления каждой переменной нажимайте **"Save"** или **"Add"**
- Для Cloudflare нужны **Email + Global API Key** (не токен!)
- Для Ukraine.com.ua нужен **только API ключ** (без секрета)

---

## 🆘 Не вижу раздел Environment?

- Убедитесь, что вы открыли именно **ваш проект** (не список проектов)
- Прокрутите меню слева вниз
- Или используйте поиск в интерфейсе Render
